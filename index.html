Economic Interpretation
  • Bull Phase: Characterized by positive average daily returns, low-to-medium rolling volatility, and contained maximum drawdowns. It represents stable upward-trending market environments.
  • Bear Phase: Marked by negative average returns, elevated rolling volatility (high uncertainty), and severe drawdowns. Standard deviation surges during these market corrections.
  • Sideways Phase: Exhibits near-zero average returns, very low rolling volatility, and minimal trend strength. Price oscillates inside horizontal trading ranges.
Quantitative Features Profile

Features are generated dynamically using the rolling lookback sliders on the sidebar. Standardization centers each feature around zero (0) mean. The clustering algorithm identifies distinct centroids in this scaled coordinate system.

PCA helps reduce collinearity among the three rolling dimensions, enabling a clean visual projection while capturing up to 80-90% of variance in the first two components.

# Python Quantitative Finance Implementation
import yfinance as yf
import pandas as pd
import numpy as np
from sklearn.preprocessing import StandardScaler
from sklearn.decomposition import PCA
from sklearn.cluster import KMeans
import scipy.cluster.hierarchy as sch

# 1. Load Data
df = yf.download("SPY", start="2023-01-01", end="2026-06-01")
df = df.reset_index()[['Date', 'Close']]

# 2. Feature Engineering
# Daily Return
df['Daily_Return'] = df['Close'].pct_change()
# Rolling Returns (21-day)
df['Rolling_Return'] = df['Close'].pct_change(periods=21)
# Rolling Volatility (21-day std dev of daily returns)
df['Rolling_Vol'] = df['Daily_Return'].rolling(window=21).std()
# Rolling Drawdown (21-day max drawdown)
rolling_max = df['Close'].rolling(window=21, min_periods=1).max()
df['Rolling_DD'] = (rolling_max - df['Close']) / rolling_max

# Drop NaNs
df = df.dropna().reset_index(drop=True)
features = ['Rolling_Return', 'Rolling_Vol', 'Rolling_DD']
X = df[features].values

# 3. Standard Scaling
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# 4. Dimensionality Reduction (PCA)
pca = PCA(n_components=2)
X_pca = pca.fit_transform(X_scaled)
print(f"Explained Variance: {pca.explained_variance_ratio_}")

# 5. K-Means Clustering
kmeans = KMeans(n_clusters=3, random_state=42, init='k-means++')
df['Regime'] = kmeans.fit_predict(X_scaled)

# 6. Hierarchical Clustering (Linkage)
# Perform Linkage on the last 150 days of scaled features
linkage_matrix = sch.linkage(X_scaled[-150:], method='average')
# The linkage matrix can be plotted using sch.dendrogram()
